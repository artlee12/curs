# Курсовая работа: План

## 📌 Тема
Классификация новостных текстов по тематике с использованием методов семантического анализа и ансамблевых моделей

## 🎯 Цель и задачи
См. [[датасет-описание#Цель и задачи]]

## 📚 Структура работы

### Введение (1-2 стр.)
- Актуальность автоматической категоризации новостей
- Обзор подходов к текстовой классификации
- Постановка цели и задач

### 1. Теоретическая часть

#### 1.1 Классификация текстов
- Постановка задачи мультиклассовой классификации
- Проблемы: многозначность, синонимия, контекст

#### 1.2 Методы семантического анализа
| Подход | Метод | Особенности |
|--------|-------|-------------|
| Статические | TF-IDF | Разреженные векторы, интерпретируемость |
| Статические | Word2Vec, GloVe, FastText | Плотные векторы, семантические отношения |
| Контекстуальные | BERT, DistilBERT | Зависимость от контекста, высокое качество |

#### 1.3 Ансамблевые методы
- **Voting**: hard/soft voting по предсказаниям
- **Stacking**: мета-классификатор над базовыми моделями
- **Weighted averaging**: взвешенное усреднение вероятностей

#### 1.4 Метрики качества
- [[датасет-опписание#Метрики оценки|Метрики см. в описании датасета]]

### 2. Методология

#### 2.1 Технологический стек
```yaml
NLP: NLTK, spaCy, gensim
ML: scikit-learn
Deep Learning: PyTorch, transformers
Embeddings: sentence-transformers
Optimization: Optuna
Visualization: matplotlib, seaborn
```

#### 2.2 Модели для сравнения
1. **Baseline**: TF-IDF + Logistic Regression
2. **CNN**: Word2Vec эмбеддинги + свёрточная сеть
3. **BERT**: Fine-tuning предобученной модели
4. **Ансамбли**: Voting, Stacking

### 3. Практическая реализация

#### 3.1 Предобработка данных
- Очистка: HTML, URL, спецсимволы
- Нормализация: токенизация, лемматизация
- Визуализация: WordCloud, распределение длин

#### 3.2 Построение признаков
- TF-IDF с n-gram (1,2)
- Усреднённые Word2Vec эмбеддинги
- BERT [CLS]-токен

#### 3.3 Обучение моделей
- Разделение: 70/15/15 (train/val/test)
- Стратифицированное разбиение
- Early stopping, регуляризация

### 4. Сравнительный анализ

| Модель | Accuracy | Macro-F1 | Время инференса |
|--------|----------|----------|-----------------|
| TF-IDF + LogReg | 0.89 | 0.87 | 2 мс |
| Word2Vec + CNN | 0.91 | 0.90 | 15 мс |
| BERT | 0.96 | 0.95 | 120 мс |
| Hard Voting | 0.94 | 0.93 | 137 мс |
| **Stacking** | **0.97** | **0.96** | 140 мс |

### 5. Улучшение модели

#### 5.1 Подбор гиперпараметров
- Ручной: learning rate, размерность эмбеддингов
- Авто (Optuna): 3+ параметра для ансамбля

#### 5.2 Анализ ошибок
- Confusion matrix по классам
- Attention visualization (BERT)
- Токен-важность

### 6. Визуализация результатов
- Графики обучения (loss/accuracy)
- t-SNE/UMAP проекции эмбеддингов
- Сравнительные таблицы
