# Курсовая работа: классификация новостных текстов

Этот репозиторий содержит материалы курсовой работы по задаче классификации новостных текстов с использованием классических ML-подходов, нейросетевых моделей и ансамблевых методов.

## Главный ноутбук

В корне репозитория лежит основной ноутбук со всем кодом:

[00_full_coursework_pipeline.ipynb](00_full_coursework_pipeline.ipynb)

В нем собран полный пайплайн работы: подготовка данных, обучение моделей, оценка качества и сохранение результатов.

## Что есть в проекте

- `curs work/` - основная рабочая директория проекта.
- `curs work/src/` - модульный Python-код для данных, моделей, обучения и утилит.
- `curs work/configs/` - конфигурации экспериментов.
- `curs work/scripts/` - скрипты для запуска обучения, предсказаний и проверки окружения.
- `curs work/tests/` - тесты для ключевых частей проекта.
- `Пояснительная_записка_Ли_Артем_Дмитриевич.pdf` - пояснительная записка к курсовой работе.
