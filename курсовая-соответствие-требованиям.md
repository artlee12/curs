## 1. Датасет, постановка задачи и метрики

**Статус:** выполнено.

Использован датасет **AG News** для многоклассовой классификации новостных текстов.

**Классы:**
- `World`
- `Sports`
- `Business`
- `Sci/Tech`

**Решаемая задача:** по тексту новости определить её тематическую категорию.

**Метрики:**
- Accuracy — основная метрика, так как классы сбалансированы.
- Precision, Recall, F1-macro — дополнительные метрики для оценки качества по всем классам.
- Confusion matrix — анализ ошибок между классами.

**Ориентиры качества:**
- Минимальная цель: Accuracy > 90%, Macro-F1 > 0.88.
- Для BERT-based моделей на AG News типичный уровень качества находится примерно в диапазоне 94-95%+ Accuracy.
- Лучший полученный результат в работе: **Stacking, Accuracy 94.17%, Macro-F1 0.942**.

## 5. Архитектура нейросетевой модели

**Статус:** выполнено.

Реализованы две нейросетевые архитектуры:

**CNN:**
- Embedding layer;
- Conv1D с kernel sizes `[2, 3, 4]`;
- Global max pooling;
- Dropout `0.3`;
- Linear classifier;
- Loss: CrossEntropyLoss;
- Optimizer: Adam.

**BERT / DistilBERT:**
- `distilbert-base-uncased`;
- sequence classification head;
- mixed precision на GPU;
- gradient accumulation;
- weight decay;
- max length 128;
- Loss: CrossEntropyLoss внутри `AutoModelForSequenceClassification`.

## 10. Подбор гиперпараметров

**Статус:** выполнено.

Выполнен автоматический подбор через **Optuna**, 30 trials.

Подбирались параметры мета-модели Stacking:
- `C`;
- `solver`;
- `max_iter`.

Лучшие параметры:
- `C = 0.02149242841367785`;
- `solver = lbfgs`;
- `max_iter = 1291`.

Результат:
- best validation F1-macro: **0.941989**;
- test Accuracy: **94.13%**;
- test F1-macro: **0.941**.

Вывод: Optuna подтвердила устойчивость stacking, но не улучшила базовый Stacking.
